Welcome to examples using Bootstrap CSS

Using CDN bootstrap is included here without the need of bower. I made this example to bring some examples on using Bootstrap on one page. It's not a reference documentation. Finally please check http://getbootstrap.com/css/. You should be able to download and view this file in your browser.

The page header is a div with class page-header and the title is a h2. You should see Bootstrap as a green label; it's just a span with two attributes: label and label-success. You can use pre-defined colors for labels using label-default, label-info, label-success, label-warning and label-danger. Same naming conventions you will find for other Bootstrap styling.

Text Alignment

I'm on the left side I'm in the middle. I'm on the right side

The alignments are done with text-center on the paragraph class, pull-left for left text span class and pull-right for the right text span class. Have a read here: http://getbootstrap.com/css/#type-alignment.

Panels

Simple Panel Title
Panels are also just div's with other CSS styles. A panel can have a title and a body represented by panel-heading and panel-body. The outter div is again handle same way as labels using classes panel and pabel-success. Panels are working smoothely with lists and tables.

The Navigation Bar

I have to say that it did cost me some nerves. The setup isn't that complicated but for dropdown you require jQuery and I included the dependencies in wrong order and wondered why the "simple" example doesn't work. However now it does work. Look at this HTML to see how it is organized. You should have a brand and then you are working with ul, li and links. The dropdown symbol is done with <span class="caret"></span>. You can adjust the navigation either to scroll it with the page or to remain visible. Tables

Tables are more or less same as you know. You just can apply a few classes to make a table look comfortable:

Style Meaning
table You see the table with the width of the page and a horizontal rule between each row.
table-condensed Consume less space in each table cell.
table-striped Provides alternating row colors
table-bordered Provides full bordering for each cell
Have a read here http://getbootstrap.com/css/#tables.
